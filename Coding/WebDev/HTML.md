#webdev 

The simplest HTML:
```html
<!DOCTYPE html>
<html>
	<head>
		<title> Title here </title>
	</head>
	<body>
		<p>Body here</p>
	</body>
</html>
```

## Common Headers

| headers | usage      | single | example                                         |
| ------- | ---------- | ------ | ----------------------------------------------- |
| h1      | header     |        |                                                 |

### List
```html
<ul>
	<li> item1 </li>

```
